RULES

        
    
          
  • Mastery = finish all items with 0 misses.
    • 
          
  • Wrong answer → hint appears → retry until correct, but accuracy keeps the miss.
    • 
          
  • On slider items, students must move every tool in the proper direction before policy buttons unlock.
    • 
          
  • The box can expand, contract, or return to no net change if opposing tools cancel each other out.
    • 
          
  • Run order is mixed so fiscal and monetary items do not all appear in one clump.
    • 
        


        


        
WHAT THIS GAME PRACTICES

        

          Fiscal policy, regulation costs and benefits, and Federal Reserve monetary tools including buying and selling securities, reserve requirements, and discount rate changes.
